---
name: analyze
description: Deep analysis of a thought — assumptions, implications, and gaps
trigger: both
parameters:
  focus:
    type: string
    description: "Analysis focus: assumptions, implications, gaps, or all"
    default: all
---

Perform a deep analysis of the current thought, focusing on: {{focus}}.

Structure your analysis as:
1. **Core Claim** — What is the thought really saying?
2. **Assumptions** — What unstated premises does it rely on?
3. **Implications** — If this is true, what follows?
4. **Gaps** — What's missing or under-explored?
5. **Strength** — How robust is this idea? What would strengthen it?

Be intellectually honest. Challenge weak reasoning but acknowledge genuine insight.
